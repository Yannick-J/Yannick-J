##  Hauptprojekt

### LeihSy – Digitales Verleihsystem
Ein webbasiertes Portal zum Verwalten und Ausleihen von Hochschul-Equipment, entwickelt im 5-köpfigen Team.

**Meine Schwerpunkte:**
- Backend-Entwicklung mit Spring Boot
- Datenbankdesign & Entity-Relationship-Modellierung
- Keycloak-Integration (OAuth2/OIDC)
- API-Design & Dokumentation
